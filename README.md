## 📌 Projects

### 🔢 Task 1 – Number Guessing Game
A console game where the user guesses a randomly generated number within a limited number of tries, with score tracking across rounds.

### 🎓 Task 2 – Student Report Card
A program that calculates a student's total marks, average percentage, and grade based on marks entered for multiple subjects.

### 🏧 Task 3 – ATM Interface
An ATM simulation allowing users to withdraw, deposit, and check balance through a simple menu-driven interface.

### 💱 Task 4 – Currency Converter
A currency conversion tool that fetches real-time exchange rates from an online API to convert an amount between two currencies.
